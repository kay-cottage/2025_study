### 📈 方法延伸：如何量化“一致性”

在统计学中，我们用 **Cohen’s Kappa (κ)** 来衡量两位诊断者的分类一致性：

$$
\kappa = \frac{P_o - P_e}{1 - P_e}
$$
*    $P_o$ ：实际一致率
*    $P_e$ ：随机一致的期望值

| κ 值范围 | 含义 |
| --- | --- |
| < 0.2 | 极低一致性 |
| 0.2–0.4 | 低 |
| 0.4–0.6 | 中等 |
| 0.6–0.8 | 良好 |
| \> 0.8 | 非常好 |

### 📘 小结：metrische Skala（度量尺度）的本质

| 特征 | 说明 |
| --- | --- |
| **连续取值** | 可用实数刻度（如 0–0.5, 0–100） |
| **可比较大小** | 具有 `<, >, =` 运算 |
| **可加减** | 可计算差值、平均值、标准差等 |
| **等距性（Äquidistanz）** | 相同差值表示相同意义 |
| **无绝对零点时 = Intervallskala** | 例如摄氏温度、年份 |
| **有绝对零点时 = Verhältnisskala** | 例如体重、长度、时间持续 |

| 尺度类型 | 示例 | 差值可比？ | 倍数有意义？ |
| --- | --- | --- | --- |

| **Intervall** | 温度(°C) | ✅ | ❌（0°C不是“无温度”） |
| **Verhältnis** | 重量(kg)、时间(s) | ✅ | ✅（10kg是5kg的两倍） |

> 🔹 实际应用中，多用于连续变量（身高、温度、浓度、得分等）。

### 🧮 MAD 的计算公式与例子

$$
\text{MAD} = 1.4826 \times \text{Median}(|x_i - \text{Median}(x)|)
$$

**例：**

数据 = \[10, 11, 12, 13, 100\]

*   Median = 12
*   |xᵢ - Median| = \[2, 1, 0, 1, 88\]
*   Median(|差|) = 1  
    → MAD = 1.4826 × 1 = **1.48**

而标准差 σ ≈ 39.5（被异常值 100 严重拉高）。  
→ MAD 对异常值更稳健。
